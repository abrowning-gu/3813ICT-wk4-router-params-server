# Overview
This is the node/express server that supports the week 4 lecture material videos. 

Once you have cloned the repo remember to run  *>npm install* to download the required dependencies.

# Running
 > node server.js
 or
 > nodemon server.js
 or
 > npm start  ( same as nodemon server.js) This is set in the scripts section of the  package.json file.
